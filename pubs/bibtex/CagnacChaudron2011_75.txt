  abstract={A consciousness-driven visual attention model is presented in this paper. It is based on a bio-inspired computer fovea model and a hierarchical analysis for the given visual receptive field. Indeed, the bio-inspired computer fovea model is used to simulate the neural activity on the human visual system, and the hierarchical analysis provides a function to explore the information on the given visual scene. The proposed model can evaluate the contents on the scene and automatically highlights visually important regions. This model can be used in various applications, such as surveillance, visual navigation and target acquisition, etc.}
